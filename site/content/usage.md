## :whale: Docker
```sh
docker run --rm -it -v $PWD:/home/nupoi ghcr.io/dangishokouta/nupoi:latest
```
The working directory in the docker container is `/home/nupoi`.
The target project should be on the directory with `-v` flag of docker.
